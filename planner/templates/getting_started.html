{# Add a new course #}
Add a new course
{# accordion-heading #}

To add a course from scratch, go to Course Summary and click the link at the top of the page.

Fill in the information for the course, and click on Submit.

Note that the "Schedule semester", "Schedule year" and "CRN" options are not currently being used, so it doesn't matter what you put there.

The course will now appear in Course Summary, but it will not yet have any Course Offerings associated with it.

Click on Manage Offerings to create one or more offerings of the new course. For further help, see Create a new offering of an existing course.

{# Create a new offering of an existing course #}
Create a new offering of an existing course
{# accordion-heading #}

To add or edit offerings of a course that already exists in the database, go to Course Summary and click on Manage Offerings.

Select the desired semester(s) for the course offering(s) and fill in the available load, the maximum enrollment and optional comments. Click on Submit.

The new course offering(s) will now show up in the Faculty Load Summary (under Combined) for the appropriate Academic Year, but no meeting times, rooms or professors will be specified.

For further information about adding meetings times, rooms or professors to a new course offering, see the appropriate help topics.

{# Add or edit course meeting time #}
Add or edit the meeting times or rooms for a course offering
{# accordion-heading #}

To add or edit the meeting times or rooms for a course offering, go to Combined on the Faculty Load Summary page and click on the appropriate link in the Time or Room column.

If the course offering had no previous offering times, the following page will appear. Choose the desired meeting day(s), time, duration and room and click on Submit.

If the course offering already had offering times and rooms, those may be edited. Fill in all entries in a given row and click on Submit.

{# Add or edit professors' loads for a course offering #}
Add or edit professors' loads for a course offering
{# accordion-heading #}

To add or edit professors' loads for a course offering, go to the Faculty Load Summary page and click on the appropriate link in the Load column.

Add or edit the load(s) for the desired professor(s) and click on Submit.

The Diff column shows the difference between the available load and the assigned load for a given course.

In the above example, there is more load assigned than is available for the course offering.

{# delete a course #}
Delete a course (completely) from the database
{# accordion-heading #}

The following describes how to completely delete a course and all of its offerings (for every year) from the database. This action cannot be undone.

Go to the Course Summary page and click on Delete .

There is one final opportunity to back out. Otherwise click Delete to seal the deal.

{# Copy a course offering into the current year #}
Copy a course offering from a previous year to the current year
{# accordion-heading #}

First go to Profile and click on Edit to choose the Academic Year to which courses will be copied. Also, Edit the Faculty for that Academic Year, as appropriate.

Then go to the Faculty Load Summary page and click on the Copy link at the top of the page.

Choose the "copy from" year.

Select courses to copy and click on Submit (at the bottom of the page -- not shown).

Note the following:

  • Greyed-out courses already exist in the "copy to" year, so they cannot be copied forward.
  • Faculty members listed in parentheses exist in the "copy from" year, but not in the "copy to" year. In this case, the course offering will be copied forward, but not the instructor.
