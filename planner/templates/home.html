Faculty Load Summary

The Faculty Load Summary page displays the course offerings being taught by the department during the given academic year, along with information such as who is teaching the course, when the course is offered, etc.

Among other things, this page allows users to:

  • Add or delete an offering of a course, or copy one from a previous year
  • Edit the semester, offering times/locations and faculty members for course offerings
  • Specify teaching and non-teaching loads for faculty members
  • Move between different academic years

Master Schedule View

The Master Schedule View displays the departmental teaching schedule in a format similar to that used in the Word document that the Registrar's office sends to department chairs during "schedule edit" season. Many types of schedule edits can be made from this page, as well.

Course Summary

The Course Summary page displays the courses taught by the department and how many offerings of each have been taught during various semesters. This page allows the user to add, edit and delete courses. (See below for the distinction between courses and course offerings.)

Courses vs. Course Offerings

Throughout the website, a distinction is drawn between courses and course offerings . A "course" has a name, a title and a certain number of credit hours. A "course offering" refers to a specific offering of a course, during a specific semester at certain time(s) during the week. There is only one course in the database corresponding to "PHY211 - University Physics," but there are several offerings of this course (typically one every fall).

The distinction between a course and a course offering is important when it comes to deleting one of them. In particular, deleting a course from the database will also delete all of the corresponding offerings, from all of the academic years. This could have unintended consequences!

Schedules

The Schedules menu has several options for visualizing where, when and by whom various course offerings are being taught.

File

The File menu contains several utilities, allowing users to:

  • Export faculty load sheets, in the (Excel) format required by Academic Affairs
  • Export department teaching and non-teaching loads as an Excel file that can be used for off-line editing and brainstorming
  • Search for courses by name and by time, including those taught in different years and by different departments

Some related screencasts:

  • Export faculty load sheets in Excel format (4:01)
  • Search for courses by name or by time (3:51)

Notes

The Notes page allows the user to add, edit and delete time-stamped notes for keeping track of ideas and to-do items regarding teaching loads and schedules.

Related screencast:

  • Use the "Notes" feature (1:39)

Help

The Help page contains links to some short screencasts that step through topics such as how to create courses, how to create, edit and delete course offerings, how to assign faculty loads, etc.

Profile

The Profile page contains information about the user, as well as the user's preferences, such as which faculty members to "view" in the various pages and which academic year to work on. These preferences can also be accessed and set from other pages throughout the website.
